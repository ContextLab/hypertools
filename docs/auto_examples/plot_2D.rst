=============================
A 2D Plot
=============================

A 2D plot can be created by setting ndims=2.




.. image:: /auto_examples/images/sphx_glr_plot_2D_001.png
    :align: center





.. code-block:: python


    # Code source: Andrew Heusser
    # License: MIT

    # import
    import hypertools as hyp

    # load data
    data = hyp.load('weights_sample')

    # plot
    hyp.plot(data, '.', ndims=2)
